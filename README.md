# EMR Suite

Modern clinical workspace built with a MEVN stack. Tailwind UI for the frontend and Express/MongoDB on the backend.

Live demo:
- https://emr-application-5i8r.onrender.com

## Quick start

Frontend:
```
cd emr-frontend
npm install
npm run dev
```

Backend:
```
cd EMR/server
npm install
npm run dev
```

## Environment

Backend:
- `MONGO_URI`
- `TWILIO_ACCOUNT_SID`
- `TWILIO_AUTH_TOKEN`
- `TWILIO_FROM`
- `EMAIL_USER`
- `EMAIL_PASS`

Frontend:
- `VITE_API_URL`
